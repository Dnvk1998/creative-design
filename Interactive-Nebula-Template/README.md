# Interactive Nebula – Sensor-Driven Canvas Template

A cinematic, interactive nebula experience built using pure HTML, CSS, and JavaScript.

## Features
- Particle nebula animation
- Text morphing
- Touch interaction (mobile)
- Gyroscope tilt flow
- Audio-reactive visuals (microphone)
- Cinematic / low-power mode
- Fully responsive
- No frameworks, no dependencies

## Live Demo
https://creative-design-xi.vercel.app

## How to Use
1. Extract the ZIP file
2. Open `index.html` in a modern browser
3. Deploy to Vercel, Netlify, or GitHub Pages

## Customization
- Change background image in `style.css`
- Control particles, audio & motion in `script.js`
- Replace images in `assets/`

## Requirements
- Modern browser (Chrome, Safari, Edge)
- Microphone access for audio-reactive mode
- Gyroscope-supported device for motion effects
